# Despachador Automático - Unidad 2

Proyecto desarrollado en Lenguaje C como parte de la Actividad 3 de la Unidad 2 de Fundamentos de Programación.

## Funcionalidades
- Menú de selección de productos
- Validación de opción incorrecta
- Opción de salida
- Implementado con estructuras de control (switch / if)

## Compilación

gcc FPR_U2_A3_ABMC.c -o FPR_U2_A3_ABMC.exe

## Ejecución

.\FPR_U2_A3_ABMC.exe
